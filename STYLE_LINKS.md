## Key CSS Patterns

### Neumorphism Shadows
```css
/* Light theme */
box-shadow: 9px 9px 16px rgba(163,177,198,0.6),
            -9px -9px 16px rgba(255,255,255,0.5);
background: #e0e5ec;

/* Dark theme */
box-shadow: 8px 8px 16px rgba(0,0,0,0.4),
            -8px -8px 16px rgba(60,60,80,0.2);
background: #2d2d3a;
```

### Glassmorphism
```css
background: rgba(255, 255, 255, 0.08-0.25);
backdrop-filter: blur(10px-20px);
-webkit-backdrop-filter: blur(10px-20px);
border: 1px solid rgba(255, 255, 255, 0.1-0.4);
```
