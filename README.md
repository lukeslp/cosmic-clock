# Cosmic Clock 2026

A mesmerizing visualization of time across multiple scales, from milliseconds to the full year cycle, with celestial events, lunar phases, zodiac constellations, and 2026 milestones.

## Features

- **Multi-scale time rings**: Minutes, hours, days, weeks, months
- **Celestial tracking**: Lunar phases, zodiac constellations
- **2026 events**: Astronomical events, sports (Olympics, World Cup), cultural holidays
- **24 stunning themes**: Bioluminescent, Glassmorphism, Skeuomorphic metals (Brass, Chrome, Cockpit, Bronze, Copper, Titanium, Rose Gold, Gunmetal, Platinum, Obsidian, Amber, Jade, Onyx), Swiss Design, Neumorphism
- **Theme-specific pulse effects**: Each visual style has a unique center pulse animation
- **Fully responsive**: Desktop and mobile optimized
- **Interactive tooltips**: Hover/tap any ring segment for details

## Technology

- Pure vanilla JavaScript + Canvas 2D rendering
- No frameworks, no build process
- Theme-aware rendering system with 24 distinct visual styles
- Real-time animation with requestAnimationFrame

## Themes

### Bioluminescent & Aurora
Multi-layered expanding pulse with organic glow effects

### Skeuomorphic Collection (14 variants)
Metallic orbs with 3D highlights and beveled edges:
- Original, Brass, Chrome, Cockpit (CRT)
- Bronze, Copper, Titanium, Rose Gold
- Gunmetal, Platinum, Obsidian, Amber, Jade, Onyx

### Swiss Design
Clean geometric pulse with precision aesthetics

### Neumorphism
Soft embossed effects with subtle shadows

### Glassmorphism
Translucent frosted glass aesthetics

## Usage

Open `index.html` in any modern browser. No server required (though file:// protocol may have CORS limitations for some browsers - use a local server for best results).

```bash
python3 -m http.server 8000
# Open http://localhost:8000
```

## Data Sources

- Astronomical events: NASA JPL, IAU
- Lunar phases: Calculated from 2026 new moon dates
- Zodiac constellations: Traditional constellation patterns
- Sports events: Official Olympic and FIFA schedules
- Cultural holidays: International holiday calendar
